Le groupe de SIMPLonMARS

Nous nous sommes occupés de superviser le groupe d'élèves et les avons aidés dans les choix de prototypes et des modifications à effectuer pour réaliser els figures demandées. Nous avons également surpervisé la création de ce site internet.

Notre aventure

La première journée nous avons dû choisir un modèle parmis les nombreux prototypes proposés.
Puis nous avons fait tous les tests afin de répondre au mieux aux figures demandés.
Nous avons décidé de sélectionner le modèle Glide puisque celui-ci nous semblait etre le modèle le plus facilement modifiable.
